Genera el diseño completo del frontend de un Portal Web Institucional moderno para una municipalidad, estilo profesional, minimalista, accesible, con enfoque UX y responsive. Debe incluir todas las pantallas, módulos, formularios, menús y secciones necesarias para que el portal funcione íntegramente. Utiliza una paleta institucional: azul oscuro, celeste, blanco, gris suave; tipografías limpias (Inter / Roboto / Open Sans), iconografía moderna y diseño alineado a estándares de gobierno digital. Inicia creando una Página de Inicio con un header institucional que incluya logo municipal, menú principal, buscador, accesibilidad (alto contraste, incrementar tamaño de texto), selector de idioma, enlaces rápidos a trámites, pagos y denuncias. Agrega un hero principal con imagen del distrito y botones CTA. Debajo, secciones destacadas: noticias recientes, acceso directo a transparencia, módulo de turismo y pesquería, calendario de eventos, enlaces a documentos públicos y banners informativos. Crear el Menú Principal con estas opciones: Inicio, Municipalidad, Noticias, Transparencia, Trámites Digitales, Pagos en Línea, Reclamos y Denuncias, Documentos Públicos, Turismo y Pesca, Participación Ciudadana, Contacto / Mesa de Partes Virtual. Diseñar cada módulo con su vista principal y subpáginas internas. Para Noticias, generar: listado con tarjetas, filtros por categoría, buscador, y una página de noticia individual con imagen, fecha, autor, contenido y documentos adjuntos. Para Transparencia, generar: dashboard con categorías (Ordenanzas, Resoluciones, Presupuesto, Documentos de Gestión, TUPA, ROF, MOF, Auditorías, Contrataciones), tabla de documentos con filtros, botones de descarga y vista previa. Diseñar el módulo de Turismo y Pesquería con secciones de atractivos turísticos, gastronomía, playas, ferias, actividades pesqueras, galería de fotos y mapa interactivo. Crear un módulo de Documentos Públicos, similar a una biblioteca digital con carpetas, buscador, filtros y vista previa de PDFs. Diseñar la sección de Participación Ciudadana: encuestas, buzón de sugerencias, audiencias públicas y consulta vecinal.

Crear todos los formularios funcionales del portal. Para Trámites Digitales, generar un formulario moderno paso a paso (wizard): datos personales, tipo de trámite, carga de documentos (PDF/imagen), verificación de datos, resumen y envío; incluir pantalla de confirmación con número de expediente. Para Reclamos y Denuncias, crear formulario con: tipo de denuncia, descripción, ubicación, adjuntos (foto/video), datos opcionales del denunciante, captcha, envío y pantalla de confirmación; agregar panel de seguimiento donde el ciudadano ingrese su código y vea el estado. Para Pagos en Línea (Culqi), crear pantalla de selección del servicio a pagar, carrito básico, formulario de datos del contribuyente, monto, integración con tarjeta, confirmación de transacción, comprobante descargable. Para Mesa de Partes Virtual, crear formulario para subir documentos oficiales, seleccionar el área destino, ingresar datos personales, adjuntar archivos, firmar digitalmente (si aplica) y obtener número de expediente.

Diseña también las pantallas del CMS para las áreas internas (solo mockups de vista, sin funciones): panel de inicio, gestión de noticias, carga de documentos, edición de páginas, vista de analytics, gestor de transparencia, control de sliders/banners y módulo de usuarios con roles. Genera también el footer institucional con dirección, horarios, enlaces legales, redes sociales y contacto. Asegúrate de que todas las pantallas sigan un mismo estilo visual, con componentes reutilizables: botones sólidos azules, hover suave, tarjetas limpias, formularios grandes y modernos, íconos claros, breadcrumbs, navegación lateral en módulos complejos, y diseño totalmente responsive para desktop, tablet y móvil. Finalmente, produce versiones: desktop, mobile y tablet de todas las vistas generadas. El resultado debe ser un set completo y consistente de mockups estáticos del portal web institucional con todos los módulos, páginas y flujos interactivos listos para desarrollo.
